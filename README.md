😷 Face Mask Detection Using YOLO

This project is a real-time Face Mask Detection system built using the YOLO (You Only Look Once) object detection algorithm. The model is trained to detect whether a person is wearing a face mask or not wearing a face mask in images and live video streams.

The project demonstrates the complete deep learning workflow, including dataset preparation, model training, inference, and evaluation. It can detect multiple faces simultaneously, making it suitable for crowded environments and real-time monitoring applications.

🚀 Features
Real-time face mask detection
Detects Mask and No Mask classes
YOLO-based object detection
Supports images, videos, and webcam input
Fast and accurate inference
Multiple face detection in a single frame
Easy-to-understand implementation

📈 Project Workflow
Prepare and label the face mask dataset.
Train the YOLO model on the dataset.
Save the trained model (best.pt).
Perform inference on images, videos, or webcam streams.
Evaluate the model using performance metrics and visualizations.
